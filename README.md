# lalalend.finance

## Getting started

### Pre-requisites

Please have these installed on your machine:

- [Node.js 12+](https://nodejs.org/)
- [Yarn v1](https://classic.yarnpkg.com/lang/)

### Install dependencies:

```
yarn
```

Then you can proceed to start the site:

```
yarn dev
```

To build the site:

```
yarn build
```
